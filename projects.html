⬢Weather API

In this project I created a python script to visualize the weather of over 500 cities using the OpenWeatherMap API. After creating a list of latitudes and longitudes for randomized cities near the equator, I then created a series of scatter plots to look for relationships between latitude and temperature, humidity, cloudiness and wind speed.


Observations

  • There is a strong correlation between a city's latitude and the maximum temperature. It is generally hotter as latitude approaches zero from either direction.
  • There is no clear correlation between a city's latitude and the percentage of cloudiness or wind speed.
  • It is hard to tell from the scatter plots if there is much correlation between a city's latitude and its humidity.
Next, the Google Places API was used to create a heat map of the cities in the list and their humidity.

Focus: data visualization; APIs; python: citipy, requests, matplotlib

View the original project on GitHub or see a website displaying the results.

⬢Logic Puzzle

This logic puzzle is sourced from Advent of Code 2021. The scenario is that we are navigating ocean depths in a submarine and that we have four-digit, seven-segment displays that are malfunctioning. Each segment in the display is powered with wires labeled with a letter a through g. Using logic and any coding language preferred, we must decipher what each of four digits would be on our display, given 10 unique signal patterns followed by our four final display combinations. There are 200 codes to crack in the dataset, so brute force is not a feasible option.


I solved this puzzle in Python, first reading the input file with csv.reader and sorting each input value alphabetically. Then, knowing that certain digits have a unique number of segments illuminated on a seven-segment display, we could solve for those digits, then use signal lengths and list subsets to solve for the remaining segments in the remaining digits to discover the four-digit output value.

Focus: logic; data processing; python
